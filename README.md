# E-Commerce Platform

A backend E-Commerce Platform built using Java 21, Spring Boot, PostgreSQL, and Maven.

## Tech Stack

* Java 21
* Spring Boot 3.x
* Spring Data JPA
* PostgreSQL
* Maven
* Docker
* Git & GitHub

## Features

### Current Features

* Spring Boot application setup
* PostgreSQL integration
* User entity implementation
* REST API foundation
* Dockerized PostgreSQL database

### Planned Features

* User Management
* Product Catalog
* Category Management
* Shopping Cart
* Order Management
* Payment Integration
* Authentication & Authorization
* Inventory Management
